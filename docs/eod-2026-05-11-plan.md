# EOD 마감 플랜 — 2026-05-11

> 5개 시나리오를 데모 가능 수준으로 굴리는 게 목표. 모든 페이지 완성이 아님.
> "준비 중" 으로 비우고 가도 시나리오에 안 걸리면 OK.

---

## 0. 베이스라인 (이미 돌아감)

- 마일리지 충전(무통장)/출금 RPC + UI + 강제 계좌 등록 다이얼로그
- listing 7단계 상태머신 (`submitted → purchased → handed_over → received → verified → shipped → completed (+cancelled)`)
- `/catalog` 공개 시세 + 매입 액션
- `/sell/new` P2P 등록
- `/buy/orders` 구매 내역 (탭 카운트 포함)
- `/admin/intake` (handed_over → received → verified → shipped)
- `/admin/mileage` 충전 confirm + 출금 처리
- `/admin/cancellations` 취소 승인/반려
- `/admin/members` agent/admin role grant
- `/agent/{inventory,sales,store}` 빈 스켈레톤 (페이지만 존재)
- 알림 RPC `notify_user / notify_users`
- AdminShell 다크 사이드바 (관리자/에이전트 공용, role 별 액센트)

각 작업 완료 시 위 체크박스 업데이트.
