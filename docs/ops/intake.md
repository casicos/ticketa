# Intake SOP (상품권 수탁)

## Steps

1. `/admin/intake` 열기
2. submitted 상태 매물 리스트 확인
3. 실물 수령 시 "수령" 버튼 클릭 → status=received
4. 진위 검수 (육안 + 발행처 전화 확인)
5. 합격 시 "검수 완료" → status=listed (자동 공개)
6. 불합격 시 사유 메모 + "취소" 버튼 → status=cancelled

## 확인 기준

- 매물 수: 판매자가 등록한 수량과 실물 수량 일치
- 시리얼: M2 진입 시 기록
- 진위: 발행처 확인번호로 대조

## 예외 처리

- 판매자 연락 안 됨: 7일 경과 후 자동 cancelled (M3 자동화, MVP는 수동)
- 실물 훼손: 사유 메모 + 취소 → 판매자에게 반송
